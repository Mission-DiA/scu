## PRODUCTION RULES — COMPLETE REFERENCE

### Gemini Prompt Requirements (Non-Negotiable)

| Rule | Description |
|------|-------------|
| **FULL BLEED** | Image extends to ALL edges, NO margins |
| **Panel Separators** | 1-2 pixel thin white lines ONLY |
| **Character Specs** | Complete descriptions in EVERY panel — NO lazy references |
| **ONE DIALOGUE RULE** | Each character gets MAX one speech bubble per panel |
| **Speech Bubbles** | Black text, white fill, black 2px outline |
| **Caption Boxes** | Dark blue (#1a1a4e) fill, white text |
| **Aspect Ratio** | 16:9 (1920x1080px) |

## NARRATIVE CONSIDERATIONS FOR S3

### The "Alphabets" Metaphor

In each universe, translate "alphabets" to that context:

| Universe | "Alphabets" = | "Real Learning" = |
|----------|---------------|-------------------|
| Prime | Academy training, certifications | Field operations, real missions |
| Engineering | CS degree, bootcamps | Production systems, oncall |
| QA | Testing certifications | Edge cases, real user behavior |
| DevOps | DevOps training programs | 3 AM outages, pipeline failures |
| SRE | Incident response training | Actual cascading failures |
| Nexus | All formal education | Professional wisdom across contexts |

### Emotional Arc for Daisy

1. **Pride** — Daisy is proud of her credentials
2. **Challenge** — Reality doesn't match training
3. **Humility** — Recognition that she's still a beginner
4. **Growth** — Embracing the learning mindset
5. **Wisdom** — Understanding that credentials were just the start

### Coulson's Role

Coulson delivers the "alphabets" message — but not condescendingly. He validates her credentials while revealing the larger truth:

> "Your training was excellent. It taught you the alphabets. Now let's learn to write."
