## OVERVIEW

This document defines the visual appearance, personality, and narrative role of all characters in the Swami Cinematic Universe. All image prompts must reference this bible for consistency.

**Version 5.0 Updates:**
- Revised Daisy costumes across all universes (removed tactical elements from tech universes)
- Revised Coulson costumes across all universes (DevOps V-neck, Nexus henley)
- Added explicit color hex codes for all costume elements
- Removed all lanyards and badges (convey roles through dialogue/narration)
- Added Viktor (DevOps Universe) with grounded, professional appearance

## VISUAL HIERARCHY RULES

### Coulson > Daisy Positioning
In ALL shared scenes:
- Coulson stands when Daisy sits
- Coulson positioned higher in frame
- Coulson has more visual "weight" (foreground, better lit)

### Character Consistency
- **SAME FACE** across all universe variants
- Only costumes and hair change per universe
- Multiverse variants are AWARE they are variants
